The let variable declaration type was created and has almost replaced every scenario you would have to use the var before now. Variables declared with the let keyword could be Updated but not Redefined. The let keyword also defines blocked scoped variables. This means that variables would only be available or accessible in between the two curly braces where it is defined and variable redeclared inside a block/child function or scope still maintains its original value when used again from the parent scope.

Variables declared with the const keyword are very similar to the variables declared with the let keyword, All the properties stated above for the let keyword is the same for the var keyword, except, Variables declared with the const keyword are constants and such, the value of a constant can't be changed through reassignment, and it can't be redeclared.
